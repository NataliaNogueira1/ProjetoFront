# Avaliação de Front-end

# *Sobre o projeto*

- O presente repositório conta com o site da **Passaraous**, uma empresa fictícia que possibilita a visualização por meio de dashboards interativos de dados relacionados ao tempo e volume de registros de postagens sobre pássaros em redes sociais.
- Após o login, o usuário é redirecionado para a homepage, no qual pode clicar em + e ver o dashboard de outras redes sociais (ficticio).
- É necessário clicar em + para abrir o pop-up e novamente no mesmo icone para fechá-lo.
- O projeto foi criado originalmente no repositório: https://github.com/NataliaNogueira1/ADS_FrontEnd.git
- A api pública consumida para o dashboard se encontra disponível em: https://jsonplaceholder.typicode.com/posts
